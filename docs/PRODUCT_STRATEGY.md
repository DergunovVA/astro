# PRODUCT_STRATEGY.md

## Vision
Build a modular, commercially viable ZET-like astrology platform
with a strong calculation engine and scalable SaaS/mobile distribution.

---

## Target Segments

### Phase 1 (Primary)
Professional astrologers
- Reliable calculations
- Configurable school modes
- Exportable charts
- Paid professional tool

### Phase 2
Advanced enthusiasts

### Phase 3
Mass audience (simplified mobile UI)

---

## Core Value Proposition

- Accurate calculations (Swiss Ephemeris-based)
- Strict school modes (Traditional / Modern / Sidereal)
- API-driven architecture
- SVG-first chart rendering
- Extensible professional toolkit

---

## Not Building Initially

- Generic AI horoscope app
- Social astrology platform
- Entertainment-only astrology

---

## Version Roadmap

v0.1 — Core Engine Stable
v0.2 — SVG Wheel + API
v0.3 — Pro Features
v1.0 — SaaS Launch
